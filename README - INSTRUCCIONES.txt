Versión de Netbeans: 12.6

El ejecutable .jar se encuentra en la carpeta dist.
Se puede ejecutar desde el CMD con el comando "java -jar Tarea2.jar + (nombre del archivo a revisar)" habiendo hecho cd a la carpeta en la que se encuentre el ejecutable.

Por ejemplo:
java -jar Tarea2.jar prueba.mingol

En la carpeta en la que se encuentre el ejecutable debe existir un archivo con el nombre ingresado y con la extensión .mingol
Puede no ingresarse la extensión .mingol al invocar el comando siempre y cuando exista un archivo mingol con ese nombre

Por ejemplo:
java -jar Tarea2.jar prueba

El comando anterior es correcto siempre y cuando exista prueba.mingol

Es necesario también que se encuentre el ejecutable del compilador Algol a68g.exe en la siguiente dirección: 
C:\Algol\a68g.exe
